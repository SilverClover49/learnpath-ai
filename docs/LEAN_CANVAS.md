# LearnPath AI - Lean Canvas

## Problem
1. **Learning overwhelm** - Too many resources, don't know where to start
2. **One-size-fits-all** - Generic courses don't match individual needs
3. **No progress tracking** - Hard to see what's been learned
4. **Lack of motivation** - No clear path or milestones

## Solution
AI-powered personalized learning platform that:
- Creates custom curricula based on goals
- Tracks progress with dynamic checklists
- Provides visual learning paths
- Celebrates achievements

## Key Metrics
- Learning plan completion rate
- Time to first milestone
- User retention (weekly active)
- Curriculum satisfaction score

## Unique Value Proposition
**"Your AI learning companion that knows you"**

Unlike generic courses, LearnPath AI creates a personalized journey that adapts to your goals, pace, and learning style.

## Unfair Advantage
- Personalized AI that learns with you
- Visual learning paths (whiteboard)
- Celebration system (end credits)
- Session persistence (remember progress)

## Channels

1. **Students** - Learning new skills
2. **Career changers** - Transitioning fields
3. **Self-learners** - Independent study
4. **Teams** - Group learning
